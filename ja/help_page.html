Mouse wheel

- Scroll: Zoom (optional switch image)

- Hold and move: Select zoom area

- Click: Zoom in/out, or change stretch mode

Left mouse

- Hold and move: Move image

- Double click: Zoom in/out, or change stretch mode

Right mouse

- Hold and move: Select crop area

- Click: Show menu

Mouse back/forward button

- Click: Previous/next image

Other useful feature

- Left/right arrow key: Previous/next image

- Drag-drop image: Then step through images in folder

- Drag-drop images: Then step through those images

- Ctrl+C, Ctrl+V: Copy/paste images from other software

- Ctrl+G: Make GIF animation from drag-droped files, or files in folder

- Ctrl+A: Set crop area to window size

- Ctrl+resize: Hold Ctrl while resizing window to preserve aspect ratio

- Shift+mouse: Toggle mouse zoom and move window behavior

- F2: Show configuration file

Touch capabilities

- Touch and move: Move image

- Two finger zoom: Zoom

- Long touch: Show menu

- Long touch and move: Select crop area

- Double tap: Zoom in/out, or change stretch mode

- Swipe right/left: Previous/next image
